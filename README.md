## Installation

#### Custom CSS Theme
First, we need to copy `custom.css` to `~/.jupyter/custom/custom.css`
```bash
cd ~/.jupyter
mkdir custom  # may already exist
cd .jupyter/custom
rm custom.css
wget https://raw.githubusercontent.com/rsomani95/jupyter-custom-theme/master/custom.css
```

#### Matplotlib Settings
Next, we gotta setup or update an IPython file that runs on starting up _any_ jupyter notebook. The contents of this file update the default `matplotlib.rcParams` settings to match the theme defined in `custom.css`. Note that the default `dpi` is set to 150 which may be a touch higher than you'd want for big visualisations.

The snippet below assumes that you do _not_ have an existing `startup.ipy` file (this is the default state). This file runs arbitrary python code and any modules imported are available in the notebook.
```bash
cd ~/.ipython/profile_default/startup
wget https://raw.githubusercontent.com/rsomani95/jupyter-custom-theme/master/startup.ipy

# If you already have a `startup.ipy` file, then you can copy paste the contents into it
```

## Further Tweaking
I applied (most of) the material theme to a template generated by the `jupyterthemes` project. The main difference is that I have parametrised a **large** chunk of the UI settings into some 80+ CSS variables available at the top of the `custom.css` file for easy tweaking.
